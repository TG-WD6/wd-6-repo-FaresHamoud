        Selector {attribute:value}
        For example:
        P { color:black:}
        p (to select <p> and apply style on it)
        color: to schoose which attribute to change, can be font-family, height, …etc)
        black; is the value we want to apply, can be px;  cm; top; bottom; …etc) -->
